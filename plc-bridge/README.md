# PLC Bridge - پلی PLC

## چییە؟
ئەم پڕۆگرامە لەسەر کۆمپیوتەرەکەت ڕایدەکەیت بۆ پەیوەستکردنی سایتەکەت بە ئامێری PLC.

**چۆن کاردەکات:**
```
سایت (ئۆنلاین) → داتابەیس → ئەم پڕۆگرامە (کۆمپیوتەرەکەت) → PLC (ئامێرەکە)
```

## هەنگاوەکان

### ١. Node.js دابمەزرێنە
لە [nodejs.org](https://nodejs.org) دایبەزێنە و دایبمەزرێنە.

### ٢. فۆڵدەری plc-bridge بکەرەوە لە Terminal
```bash
cd plc-bridge
```

### ٣. فایلی .env دروست بکە
`.env.example` کۆپی بکە بۆ `.env` و IP ـی PLC ـەکەت بنووسە:
```bash
cp .env.example .env
```
ئینجا فایلی `.env` بکەرەوە و `PLC_IP` بگۆڕە بۆ IP ـی ئامێرەکەت.

### ٤. پاکەجەکان دابمەزرێنە
```bash
npm install
```

### ٥. پڕۆگرامەکە ڕابکە
```bash
npm start
```

ئەگەر هەموو شت ڕاست بێت، دەبینیت:
```
╔══════════════════════════════════════════╗
║       🤖 PLC Bridge - پلی PLC          ║
╠══════════════════════════════════════════╣
║  PLC IP:    192.168.1.100               ║
║  PLC Port:  502                         ║
╚══════════════════════════════════════════╝

✅ PLC پەیوەستە! (Modbus TCP)
✅ داتابەیس پەیوەستە!
👀 سەیرکردنی ئۆردەرەکان دەستی پێکرد...
```

### ٦. تاقیکردنەوە
لە سایتەکەت ئۆردەرێک بدە. لە terminal ـەکە دەبینیت:
```
🔔 1 ئۆردەری نوێ دۆزرایەوە!
📦 ئۆردەر #001 - 2 ئایتم
   🔧 Espresso | PLC Code: 1 | Qty: 1
   ✅ نێردرا بۆ PLC
```

## گرنگ
- کۆمپیوتەرەکەت دەبێت لە **هەمان نێتوۆرکی** PLC بێت
- پڕۆگرامەکە دەبێت **هەمیشە ڕووە بێت** کاتێک ئامێرەکە کاردەکات
- بۆ ڕوونگەلەکردنی هەمیشەیی، دەتوانیت `pm2` بەکاربهێنیت:
  ```bash
  npm install -g pm2
  pm2 start bridge.js --name plc-bridge
  pm2 startup
  pm2 save
  ```
